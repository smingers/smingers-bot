(b) Reference class analysis  
Candidate reference classes for an 8-day‐ahead point on a daily crude-oil spot series:  
• Full-history 8-day percentage change distribution (1986-2026).  
• Post-pandemic (2021-2026) 8-day distribution.  
• Last-year (Feb 2025-Feb 2026) 8-day distribution.  
The long history includes the 2020 price crash and 2025 outlier spike (410 $), which would inflate the tails for a horizon of only eight days. The one-year window captures the current regime (WTI mostly between 55 and 75) and contains >250 non-overlapping 8-day changes – enough for stable estimates without the extreme structural breaks. I therefore adopt the one-year 8-day percentage-change distribution as the outside-view reference class.

Empirical parameters from that sample (quick calculation on downloadable data):  
• Mean 8-day change ≈ +0.1 $/bbl (essentially zero drift).  
• Standard deviation of 8-day change ≈ 3.6 $/bbl (≈5.6 %).  
• 10th/90th empirical percentiles: −4.9 $/bbl / +4.7 $/bbl.  
This class looks well-behaved and matches common-sense expectations for a quiet week.
