The site aggregates GB polling data using a weighted average system with the following key features:

1. **Pollster Accuracy Weighting:** Each polling firm is assigned a score based on its historical accuracy relative to actual election results. Firms with stronger track records carry more influence on the trendline than those with higher historical error margins.

2. **Frequency Adjustment:** To prevent high-volume pollsters from disproportionately skewing the average, weights are adjusted for publication frequency.
